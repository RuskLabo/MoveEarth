## サーバーの内容

#### Towny

1:500スケールのアースマップで国を立てたり戦争をすることができます

#### ディメンション

アース、ネザー、エンドの4つのディメンションで国家を立てたり戦争したりできます

#### Movecraft

自分の好きな 航空機や地上車両,海上船舶を作って遊ぶことができます
